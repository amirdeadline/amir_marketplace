# Approval Required

Review the commands, risks, expected impact, and rollback plan above.

Reply with one of the following:

1. `Approve plan` — run the plan exactly as written.
2. `Approve actions 1-3` — approve only specified actions (adjust numbers).
3. `Revise plan: <instructions>` — change the proposed plan.
4. `Cancel` — make no changes.

No state-changing command will run without explicit approval.

**Not approval:** Okay, Interesting, Continue investigating, What do you think?, Maybe, silence, or unrelated replies.
